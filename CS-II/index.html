CS-II

August 2021 - May 2022 : Advanced Computer Science II

  • In Computer Science II (CS II), I learned about a variety of Computer Science topics, including APIs, HTML/CSS/JavaScript, Lindenmayer Systems, software design patterns, and more.
  • An example of an API we studied was REST API, an API widely used to facilitate communications on the web due to its useful requirements, such as being stateless or having cacheable data.
  • While making our Arena Scheduling Webpage, we used HTML to form the basic outline of our website, CSS to style our website, and JavaScript to add additional interactivity.
  • We also learned about how Lindenmayer Systems could recursively update strings to produce the instructions to draw self-similar patterns such as digital trees.
  • Finally, a software design pattern we learned about was Model-View-Controller, a pattern which divides a program into a model, which holds data, a view, which relates to what the user sees, and a controller, which processes inputs from the view and updates the model.
