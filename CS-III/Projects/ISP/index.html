ISP

Our project, "FreeCash" is a Low-Frequency Trading AI, is a service designed to give reliable prediction data for given stocks. It uses 3 indicators: Twitter sentiment data, Google search data, and r/wallstreetbets sentiments. Using these 3 indicators it will create an index which will then be used with thousands of past market data to train an AI via reinforcement learning. Once this AI has been trained to score above 50% accuracy resulting in a net profit, it will be released to the public as a subscription service.
